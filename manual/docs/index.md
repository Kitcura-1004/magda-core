# Welcome to MAGDA

MAGDA (**M**ulti **A**gent **D**igital **A**udio) is a free, open-source digital audio workstation (DAW) for recording, editing, and mixing audio and MIDI. Built with [JUCE](https://juce.com/) and [Tracktion Engine](https://github.com/Tracktion/tracktion_engine), it provides a professional-grade audio engine in a lightweight, open-source package.

MAGDA includes a built-in **AI assistant** — describe what you want in natural language ("add a MIDI track with a bass clip", "transpose the selected notes up an octave") and MAGDA translates your request into actions. See [AI Assistant](panels/ai-assistant.md) to learn more.

## Three Views, One Workflow

MAGDA is organized around three main views, each focused on a different stage of music production. Switch between them using the mode buttons in the footer bar.

### ![Session](assets/images/icons/Session.svg){ width="24" } Session View (Live Mode)

A grid-based clip launcher for real-time performance and idea sketching. Trigger clips and scenes on a grid, with a mini mixer strip at the bottom of each track.

[Go to Session View →](session-view.md)

### ![Arrangement](assets/images/icons/Arrangement.svg){ width="24" } Arrangement View (Arrange Mode)

A traditional timeline for composing, recording, and editing. Lay out clips on tracks along a horizontal timeline with full zoom, selection, and editing tools.

[Go to Arrangement View →](arrangement-view.md)

### ![Mix](assets/images/icons/Mix.svg){ width="24" } Mixer View (Mix Mode)

A channel-strip mixer for balancing levels, panning, routing, and adding effects. Each track gets a full-height fader with metering, I/O routing, and send slots.

[Go to Mixer View →](mixer-view.md)

## Key Concepts

- **[Tracks](tracks.md)** — Hybrid tracks that accept any clip type, with FX chains and racks for flexible routing
- **[Built-in Devices](devices/built-in.md)** — Drum Grid, Sampler, synths, and effects included out of the box
- **[Modulation](modulation/overview.md)** — LFOs, bezier curve shapes, and macros for dynamic parameter control
- **[Interface Layout](interface/overview.md)** — Collapsible panels, context-sensitive editors, and plugin/media browsers
