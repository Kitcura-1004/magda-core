# Welcome to MAGDA

MAGDA is a free, open-source digital audio workstation (DAW) for recording, editing, and mixing audio and MIDI. It features a built-in AI assistant that understands natural language — describe what you want ("add a MIDI track with a bass clip", "transpose the selected notes up an octave") and MAGDA translates your request into actions. Built with [JUCE](https://juce.com/) and [Tracktion Engine](https://github.com/Tracktion/tracktion_engine), it provides a professional-grade audio engine in a lightweight, open-source package.

## Three Views, One Workflow

MAGDA is organized around three main views, each focused on a different stage of music production. Switch between them using the mode buttons in the footer bar.

### Session View (Live Mode)

An Ableton Live-style clip launcher for real-time performance and idea sketching. Trigger clips and scenes on a grid, with a mini mixer strip at the bottom of each track.

[Go to Session View →](session-view/overview.md)

### Arrangement View (Arrange Mode)

A traditional timeline for composing, recording, and editing. Lay out clips on tracks along a horizontal timeline with full zoom, selection, and editing tools.

[Go to Arrangement View →](arrangement-view/overview.md)

### Mixer View (Mix Mode)

A channel-strip mixer for balancing levels, panning, routing, and adding effects. Each track gets a full-height fader with metering, I/O routing, and send slots.

[Go to Mixer View →](mixer-view/overview.md)
