# Keyboard Shortcuts

## Transport

| Shortcut | Action |
|----------|--------|
| ++space++ | Play / Stop |
| ++r++ | Record |
| ++home++ | Return to start |

## Editing

| Shortcut | Action |
|----------|--------|
| ++ctrl+z++ / ++cmd+z++ | Undo |
| ++ctrl+shift+z++ / ++cmd+shift+z++ | Redo |
| ++ctrl+x++ / ++cmd+x++ | Cut |
| ++ctrl+c++ / ++cmd+c++ | Copy |
| ++ctrl+v++ / ++cmd+v++ | Paste |
| ++ctrl+d++ / ++cmd+d++ | Duplicate clips or track |
| ++ctrl+shift+d++ / ++cmd+shift+d++ | Duplicate track (without content) |
| ++ctrl+a++ / ++cmd+a++ | Select all |
| ++delete++ / ++backspace++ | Delete selected clips or track |
| ++ctrl+e++ / ++cmd+e++ | Split / trim at cursor or selection |
| ++ctrl+j++ / ++cmd+j++ | Join clips |
| ++ctrl+b++ / ++cmd+b++ | Render clip to audio |
| ++ctrl+shift+b++ / ++cmd+shift+b++ | Render time selection |

## Tracks

| Shortcut | Action |
|----------|--------|
| ++ctrl+t++ / ++cmd+t++ | Add new track |
| ++m++ | Toggle mute on selected track |
| ++shift+s++ | Toggle solo on selected track |

## Navigation

| Shortcut | Action |
|----------|--------|
| ++ctrl+plus++ / ++cmd+plus++ | Zoom in |
| ++ctrl+minus++ / ++cmd+minus++ | Zoom out |
| ++ctrl+shift+l++ / ++cmd+shift+l++ | Set loop from clip bounds |

## File

| Shortcut | Action |
|----------|--------|
| ++ctrl+s++ / ++cmd+s++ | Save |
| ++ctrl+shift+s++ / ++cmd+shift+s++ | Save as |

## Other

| Shortcut | Action |
|----------|--------|
| ++escape++ | Exit link mode / clear edit cursor |
